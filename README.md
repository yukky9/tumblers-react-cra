# Современный сайт ОАО «Смоленский завод радиодеталей»

React + TypeScript + Tailwind CSS версия сайта с сохраненной структурой исходника и современным дизайном. Проект сделан в обычной React / Create React App структуре на `react-scripts`, без Vite.

## Что внутри

- адаптивная шапка с разделами: каталог, продукция и услуги, информация, прайс-лист, контакты, вакансии;
- главная страница с hero-блоком, каталогом, блоком «О нас», услугами, информацией для акционеров и CTA;
- страницы каталога, услуг, аренды, прайса, контактов, вакансий, отчетов, партнеров, реквизитов и закупок;
- модальное окно заявки / консультанта;
- карточки продукции, услуг, документов и вакансий;
- Tailwind-дизайн в индустриальном стиле: темный hero, сетка, стеклянные карточки, акцентный зеленый цвет;
- стандартная CRA-сборка в папку `build`.

## Запуск

```bash
npm install
npm start
```

По умолчанию сайт откроется на `http://localhost:3000`.

## Сборка

```bash
npm run build
```

Готовые статические файлы будут в папке `build/`.

## Где менять контент

Основной контент вынесен в `src/data.ts`: навигация, категории продукции, услуги, контакты, вакансии, документы и ссылки на PDF.

## Что подключить для production

- реальные PDF-ссылки на годовые / финансовые отчеты и СОУТ;
- backend или почтовый шлюз для формы заявки;
- карту на странице контактов;
- CMS/API для каталога и прайс-листа, если данные должны обновляться без пересборки сайта.
